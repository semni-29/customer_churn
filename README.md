## 2. 프로젝트 목표
### 1) 이탈 고객 특성 분석을 통해 이탈 방지 전략 구축 단계
  - 이탈 고객 분석 : 이탈 고객을 분석하여 공통 특성 파악
  - 타겟 세그멘테이션(고객세분화) : 이탈 위험 고객을 세분화하고 핵심 타겟 설정
  - 맞춤화 전략 기획 : 타겟의 니즈와 특성을 반영한 맞춤화 전략을 기획하고 실행
  - 개인화된 렌탈 경험 : 이탈 사유 개선 및 개인화된 렌탈 경험 제공을 통한 소비자 경험 향상
